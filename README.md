# Python Expense Tracker

This is a beginner-friendly Python script for tracking personal expenses. 
It allows users to:

- Add expenses with name, amount, and category
- View all expenses in a table
- See the total amount spent

## Technologies Used
- Python
- CSV module
- File I/O

## How to Use
1. Run `expense_tracker.py` in a Python environment.
2. Follow the menu to add or view expenses.
3. Expenses are saved in `expenses.csv`.
